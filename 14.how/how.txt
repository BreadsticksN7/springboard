In your own terms, define the following terms:

What is HTTP?
    Hyper text transfer protocol.  Defines rules on how web clients send and receive data

What is a URL?
    Address for an internet somesite

What is DNS?
    System that translates IP information into urls

What is a query string?
    Allows you to search for values in the urls

What are two HTTP verbs and how are they different?
    Get / Post - get requests information, post sends

What is an HTTP request?
    A request from the client that follows HTTP protocols

What is an HTTP response?
    A response from the server that follows the HTTP protocols

What is an HTTP header? Give a couple examples of request and response headers you have seen.
    Provides information about the request or response
    Host / User Agent / Cookies

What are the processes that happen when you type “http://somesite.com/some/page.html” into a browser?
    Client browser translates the url into an IP address from a DNS server to resolve the address.  Request sends
    the header information back to the client.  Server creates the content requested by the client.
